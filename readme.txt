== Description ==

Questo plugin mostra gli ultimi post tra 3 temi di Napolike: eventi, sagre e viabilità.
Napolike è il più grande portale sulla città di Napoli, con eventi, turismo e informazioni su servizi pubblici e traffico.

Con questo widget tutti possono fornire un servizio utile ai propri utenti, pensate ad un hotel che non ha modo di dire agli utenti cosa possono fare nella nostra città o come muoversi, o cosa non possono assolutamente perdere.

Semplice da usare, basta installarlo, aggiungere ai widget e selezionare l'area di interesse preferita.


Questo plugin utilizza una funzionalità di terza parte per mantenere le limitazioni CORS sul lato server, solo durante le nostre richieste ajax.
http://server1.madeit.srl

Un duplicato di:
https://cors-anywhere.herokuapp.com/

Questa funzionalità non terrà traccia del vostro utilizzo, né caricherà alcuno script sulla vostra pagina.

-----------

English

This plugin shows, only in Italian language, the latest posts of three tematics from Napolike.
Napolike is the biggest website to the city of naples, with events, tourism and infos on public services, roads and traffic.

With this widget everyone can give a useful service to their users, think about an hotel that has no way to tell users what they can do in our city or how to move, or what they must see.

Simply to use, just install, add to your widgets and select your favourite area of interest.

This plugin uses a third part functionality to keep CORS limitations on the server side, only during our ajax requests.
http://server1.madeit.srl

A duplicate of
https://cors-anywhere.herokuapp.com/

This functionality will not keep track of your usage, neither load any script on your page.

There you can find all the information about this service:
Source code   :   https://github.com/Rob--W/cors-anywhere/
Documentation :   https://github.com/Rob--W/cors-anywhere/#documentation
Privacy: https://www.heroku.com/policy/promise

== Installation ==
 
1. Extract npk-widget.zip
2. Upload `napolike-widget` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
